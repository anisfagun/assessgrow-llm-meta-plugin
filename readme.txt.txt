Adds custom <meta> tags to all WordPress pages for AI disclosure and SEO optimization. Built for AssessGrow.

== Description ==

This plugin injects meta tags sitewide via wp_head for:

- AI content disclosure
- Search engine instructions
- Transparency with users and crawlers

Useful for sites that use AI-enhanced or workplace-adapted content, such as AssessGrow’s high-difficulty assessment platform.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the WordPress dashboard
3. Meta tags will be automatically added to all front-end pages

== Frequently Asked Questions ==

= Does this plugin require settings? =  
No. It works immediately upon activation. No configuration needed.

= Can I modify the meta tag content? =  
Yes, you can edit the PHP file directly or use a filter hook (if added later).
